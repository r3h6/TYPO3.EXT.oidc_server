OpenID Connect server for TYPO3 based on `OAuth 2.0 OpenID Connect Server <https://github.com/steverhoades/oauth2-openid-connect-server>`__.
Use TYPO3 as Identity Provider (IdP) or Resource Server (RS) for Single Sign-On (SSO).
